# StadiaAI --- Antigravity Master UI Prompt

## Product

**Name:** StadiaAI

**Tagline:** *The AI Operating System for Smart Stadiums*

------------------------------------------------------------------------

## Overview

StadiaAI is a next-generation AI platform built for the FIFA World Cup
2026 to improve the experience of fans, organizers, volunteers, venue
staff, security teams, medical responders, and transportation operators.

This is **not** a sports streaming app. It is an AI-first stadium
operations platform that combines conversational AI, real-time
operational intelligence, navigation, crowd management, accessibility,
multilingual assistance, and decision support into a single product.

Design it as if it were an official FIFA deployment.

------------------------------------------------------------------------

# Design Goals

Create a premium, production-ready UI inspired by:

-   Apple Human Interface
-   Linear
-   Stripe Dashboard
-   Raycast
-   Formula 1 Race Control
-   Vercel

The experience should feel calm, futuristic, elegant, and data-rich.

------------------------------------------------------------------------

# Visual Language

-   Dark theme
-   Glassmorphism
-   Soft blue/cyan glows
-   Rounded 24--32px cards
-   Minimal but information-dense
-   Smooth gradients
-   Floating navigation
-   Premium typography
-   Motion-ready components

Avoid clutter and unnecessary decoration.
